Use Array Callbacks
Callbacks & Higher-Order Functions
⬜⬜ Intermediate
You have an array of numbers. Use array methods with callbacks to:

1. doubled — every number multiplied by 2 (use .map())
2. evens — only the even numbers (use .filter())
3. total — the sum of all numbers (use .reduce())
💡 Hint: .reduce((sum, n) => sum + n, 0) — the 0 is the starting value

// Output will appear here...
// Reference
const nums = [1, 2, 3]; nums.map(n => n * 2); // [2, 4, 6] nums.filter(n => n > 1); // [2, 3] nums.reduce((sum, n) => sum + n, 0); // 6

Build a Closure
Closures & Scope
⬜⬜⬜ Challenge
Create a function called makeMultiplier that takes a number factor and returns a new function. The returned function should take a number n and return n * factor.

Then use it to create triple (factor 3) and half (factor 0.5).
💡 Hint: function makeMultiplier(factor) { return n => n * factor; }

// Output will appear here...
// Reference
// A closure: inner fn remembers outer variable function makeCounter() { let count = 0; return () => ++count; // remembers `count` } const counter = makeCounter(); counter(); // 1 counter(); // 2
